Constructor detail
Spheroid3d()constructor
public function Spheroid3d(target:DisplayObjectContainer, width:Number, height:Number, depth:Number, x:Number = 0, y:Number = 0, z:Number = 0, rotation:Number = 0, rotationY:Number = 0, rotationZ:Number = 0, jitterX:Number = 0, jitterY:Number = 0, jitterZ:Number = 0)

Distributes nodes in a 3d spheroid.

Parameters
target:DisplayObjectContainer — DisplayObjectContainer which parents all objects in the layout
 
width:Number — Width of the spheroid
 
height:Number — Height of the spheroid
 
depth:Number — Depth of the spheroid
 
x:Number (default = 0) — x position of the spheroid
 
y:Number (default = 0) — y position of the spheroid
 
z:Number (default = 0) — z position of the spheroid
 
rotation:Number (default = 0) — Rotation of the spheroid along the x-axis
 
rotationY:Number (default = 0) — Rotation of the spheroid along the y-axis
 
rotationZ:Number (default = 0) — Rotation of the spheroid along the z-axis
 
jitterX:Number (default = 0) — Jitter multiplier for the layout's nodes on the x axis
 
jitterY:Number (default = 0) — Jitter multiplier for the layout's nodes on the y axis
 
jitterZ:Number (default = 0) — Jitter multiplier for the layout's nodes on the z axis

Example 
   _spheroid=new Spheroid3d(this, 150, 150, 150, 400, 400, 0);
   
   var s:Shape;
   for(var i:int=0; i<100; i++)
   {
     s = new Shape();
     s.graphics.beginFill(0);
     s.graphics.drawCircle(0, 0, 3);
     s.graphics.endFill();
     
     _spheroid.addToLayout(s, false);
   }
   
   _spheroid.distributeNodes();
   _spheroid.updateAndRender();

distributeNodes()method 
public function distributeNodes(iterations:int = 25):void

Attempted to create an even pattern of nodes across the surface of the spheroid. The higher the number of iterations, the more accurate. Note - The processing time increases exponentially the higher the iterations go.

Parameters
iterations:int (default = 25) — The number of passes to create an even distribution of nodes across the spheroid
