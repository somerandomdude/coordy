ALLOW_OVERFLOWconstant
public static const ALLOW_OVERFLOW:String = "allow"

Allow all nodes within flow that are out of the layout's bounds to continue flowing. For example, a flow layout in which the flowDirection was HORIZONTAL would continue places nodes horizontally even if the bounds of the layout have been reached
